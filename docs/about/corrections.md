# Corrections

A public log of every correction applied to charts and published numbers
on this site.

## The commitment

> *"Every headline number on the site is reproducible, traceable to a
> primary regulator source, and survives hostile reading."*

When a number on this site is wrong, we correct it in public. This page
records every correction with the date it was applied, the chart or page
affected, the nature of the issue, the resolution, and the commit SHA
that closed it.

## How to file a correction

Open a GitHub issue with the `correction` label:

[github.com/richardjlyon/uk-subsidy-tracker/issues?q=is%3Aissue%20label%3Acorrection](https://github.com/richardjlyon/uk-subsidy-tracker/issues?q=is%3Aissue%20label%3Acorrection)

When filing, include:

- The chart, page, or published number in question.
- A primary source (regulator publication, dataset URL) supporting the
  correction, not a secondary commentary.
- The magnitude of the error if you can estimate it.

Substantive corrections (changing a headline number by more than 1%)
trigger an entry in [`CHANGES.md`](https://github.com/richardjlyon/uk-subsidy-tracker/blob/main/CHANGES.md),
and formula-level corrections bump the `methodology_version` string in
the gas counterfactual module.
