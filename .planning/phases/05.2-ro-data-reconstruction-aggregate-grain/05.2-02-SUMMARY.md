- Verification method: extracted XLSX cell values → computed buyout = worth − recycle → compared against roc-prices.csv rows

### SY5-SY8 Derivation (Traceable from Primary Sources)

- **Formula:** `recycle_gbp_per_roc = total_redistributed / total_rocs_issued`
- **Numerator source:** SY18 XLSX "4. Supplier Compliance" Table 4.5 — redistribution amounts 2006-07 to 2009-10
- **Denominator source:** 12-year XLSX "ROCs by Tech & Country" annual-section TOTAL rows — annual ROC issue counts
- **Cross-check:** SY9 recycle (14.35) matches Figure A4.1 exactly; SY5-SY8 uses same formula → <0.1% drift expected
- **Statutory buyout prices:** UK SI instruments (SI 2006/3270, 2007/3086, 2008/3107, 2009/2982)
- This derivation is methodology-compliant: auditors can recompute by dividing redistribution totals by ROC issuance counts from the same two XLSX files.
