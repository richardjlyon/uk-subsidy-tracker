## SY17 Status

The wave_2_handoff_facts stated "SY17 IS in ro-generation.csv (the 12-year XLSX naturally carries SY17 months Apr 2018 – Mar 2019)." This is **factually incorrect**. The 12-year XLSX covers Apr 2006 – Mar 2018 = SY5 through SY16. SY17 (Apr 2018 – Mar 2019) is absent from both sources:

- 12-year XLSX: max data is Mar 2018 (month=3, year=2018) → SY16
- ro-annual-aggregate.csv: starts at SY18 (year=2020)
- Gap: SY17 (year=2019) is the PDF-only deferred year per REQUIREMENTS.md

Consequence: annual_summary.parquet has no row for year=2019. This is the correct behaviour — SY17 data is deferred pending PDF transcription (backlog item). Plan 06 REF reconciliation will need to handle this gap.
