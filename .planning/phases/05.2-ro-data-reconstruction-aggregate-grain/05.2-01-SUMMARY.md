## Decisions Made

- **`sources=None` ⇒ key omitted (not `"sources": null`)** — preserves byte-parity for the 8 existing single-URL sidecars across CfD, ONS gas SAP, Elexon, and the Phase 5 Option-D RO stubs. A serialised `null` would have invalidated `test_manifest` round-trip determinism on every prior sidecar without code change. Verified by `test_write_sidecar_sources_none_byte_parity_with_no_kwarg` (post-strip-`retrieved_at` parity assertion).
- **`parse_xlsx_to_monthly()` is a NotImplementedError stub, not a no-op** — accidental refresh-loop invocation fails loud rather than silently emitting an empty Parquet that would break Wave-3 verification. The signature + docstring + return-shape contract (`ofgem_monthly_schema`) are pinned now so Wave 3 lands as a body-only edit.
- **D-17 download pattern copied verbatim from `data/ons_gas.py`** — `output_path` bound BEFORE `try:` (gap #2 fix from Phase 4 Plan 07), `timeout=60`, bare `raise` in `except RequestException`. The pre-existing `data/lccc.py` `except` shape (silent return) is the deprecated pattern per Phase 4 D-17.
- **Standardised skip reason `"dormant per Phase 05.2; un-skip on backlog 999.1"` is the single grep-discoverable switch** — both registered in conftest.py `pytest_configure` (marker description) and emitted by `pytest_collection_modifyitems` (auto-skip reason). Matches D-06 contract verbatim.

The literal `raise$` pattern doesn't survive the inline comment; the discipline does.
